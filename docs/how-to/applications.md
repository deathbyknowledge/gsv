# Applications

Applications are packages anyone can write, share, and run on GSV. Before a package runs, it goes through review — you can have one of your agents review it, or review it yourself — so nothing executes until you approve it.

## Install an application

1. Open **Applications** and click **New application.**
2. Paste the package link and click **Import application.**
3. Choose a reviewer: one of your agents, or yourself.
4. When the review is approved, click **Approve & enable.**
5. Click **Open app.**

## Try it: Starfield

Import `https://github.com/deathbyknowledge/starfield` and try out our GSV made Starfield app :).
