# Applications

Applications are packages anyone can write, share, and run on GSV. Before a package runs, it goes through review by one of your agents, then waits for your approval, so nothing executes until you enable it.

## Install an application

1. Open **Applications** and click **New application.**
2. Paste the package link and click **Import application.**
3. Choose a reviewer agent, or keep the default personal agent.
4. When the review is approved, click **Approve & enable.**
5. Click **Open app.**

## Try it: Starfield

Import `https://github.com/deathbyknowledge/starfield`, then fly around with **WASD** to explore.
