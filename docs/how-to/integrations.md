# Integrations (MCP)

Integrations give GSV new tools through [MCP](https://modelcontextprotocol.io) servers — calendars, search, your own internal services, anything that speaks MCP.

## Add an MCP server

1. Open **Integrations.**
2. Add a new server and provide its connection details.
