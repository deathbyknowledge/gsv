# Settings

[Back to GSV Manual](../../index.md)

Settings control how GSV behaves for the whole system, for a user, and for agents. Start with curated settings in the GSV console. Use raw config only for recovery or advanced troubleshooting.

## Main Settings Areas

- System settings: server behavior, runtime defaults, and installation-wide options.
- User preferences: account-level choices and overrides.
- AI model and provider settings: model choice, API provider, generation behavior, and context limits.
- Voice and transcription settings: speech output and audio-to-text behavior.
- Approval behavior: when tools ask the user, run automatically, or are denied.
- Auth and session settings: login, setup, session lifetime, and connection behavior.
- Raw config recovery: direct inspection and repair when a normal setting cannot be used.

## For Agents

Changing settings affects future work. State which setting you changed, whether it is system-wide or user-specific, and how to undo it if the user asks.
